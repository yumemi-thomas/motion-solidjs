# delay

<Todo>
  solidjs-motion doesn't re-export `delay`; import it from the engine: `import {delay} from
  'motion-dom'`.
</Todo>

`delay` is a cancellable `setTimeout` scheduled on motion's frame loop, so it stays aligned with
animation rendering.

<Example slug="delay-fn" />

`delayInMs` is exported as an alias for files that already have another `delay` in scope.
