# createTime

Returns a `MotionValue<number>` that updates every frame with `performance.now()`-style timestamps.
Combine it with [`createTransform`](/docs/create-transform) for time-based loops.

```tsx
import { createTime, createTransform, motion } from 'solidjs-motion'

function Spinner() {
  const time = createTime()
  const rotate = createTransform(time, [0, 4000], [0, 360], { clamp: false })
  return <motion.div style={{ rotate }} />
}
```
