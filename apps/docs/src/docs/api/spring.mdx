# spring

`spring({...})` returns a generator function you can pass directly to `animate()` or a `motion.*`
transition. Use it when you need direct control over spring physics beyond
`transition={{ type: 'spring' }}`.

```tsx
import { animate, spring } from 'motion-solidjs'

animate(element, { x: 200 }, spring({ stiffness: 280, damping: 18 }))
```

## Options

- `stiffness` (default 100)
- `damping` (default 10)
- `mass` (default 1)
- `velocity` - initial velocity
- `bounce`, `duration` - declarative parameterization
