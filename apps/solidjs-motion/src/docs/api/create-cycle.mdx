# createCycle

<Todo>
  solidjs-motion has no `createCycle`. It's a few lines over a `createSignal` — see the example.
</Todo>

`createCycle` returns `[state, cycle]` for stepping through a fixed set of values. Calling
`cycle()` advances to the next value; `cycle(i)` jumps to a specific index.

```tsx
import { createCycle, motion } from 'solidjs-motion'

const variants = {
  closed: { opacity: 0, y: -10 },
  open: { opacity: 1, y: 0 },
}

function Menu() {
  const [state, cycle] = createCycle('closed', 'open')
  return (
    <>
      <button onClick={() => cycle()}>Toggle</button>
      <motion.nav animate={state()} variants={variants} />
    </>
  )
}
```

## API

- `createCycle<T>(...values: T[])` - pass two or more values
- Returns `[state, cycle]`
- `cycle()` - move to the next item and wrap at the end
- `cycle(i)` - jump to index `i`
